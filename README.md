## üé® Customization Guide

### Update Personal Information

Replace these in `index.html`:
- Navigation logo and title
- Hero section name and description  
- About section biography
- Skills with your tech stack
- Projects with your work
- Contact links (email, GitHub, LinkedIn)

### Modify Colors

Main color scheme in `styles.css`:
```css
/* Primary gradient */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Text colors */
color: #2c3e50; /* Dark headings */
color: #555;    /* Body text */
```

## üöÄ Deployment on GitHub Pages

### Quick Deploy Steps:

1. **Create GitHub Repository**
   ```bash
   # Initialize git (if not already done)
   git init
   git add .
   git commit -m "Initial portfolio commit"
   ```

2. **Push to GitHub**
   ```bash
   git remote add origin https://github.com/Jaskirat12206/[repo-name].git
   git branch -M main
   git push -u origin main
   ```

3. **Enable GitHub Pages**
   - Go to repository Settings
   - Navigate to Pages (left sidebar)
   - Source: Deploy from branch
   - Branch: `main`
   - Folder: `/ (root)`
   - Click Save

4. **Your Live Site**
   - URL: `https://jaskirat12206.github.io/[repo-name]`
   - Usually live in 2-5 minutes
